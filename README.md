# art-resources-map

## Overview 
The goal of this website is to curate a collection of art resources
meant for beginners, intermediate, or advanced artists. This
will be a manually created database. Some resources I have
personally used, while others have been recommended by online
users on social media. These resources are comprised
of videos, books, and online articles. Resources will be paid or
free. 

## Technology

### Front-end
React, Node.js, JavaScript, HTML, CSS, TailwindCSS

### Back-end
Express.js, JavaScript
